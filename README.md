# 🚴‍♂️ Bike Trails — Веломаршруты для райдеров

Интерактивный каталог велосипедных маршрутов с фильтрацией по сложности. Проект создан для демонстрации возможностей HTML, CSS и JavaScript.

## ✨ Возможности

- 🗺️ 6 готовых маршрутов с описанием
- 🟢🔵⚫ Фильтрация по уровню сложности (лёгкие / средние / сложные)
- 📊 Детальная информация: длина, набор высоты, рейтинг
- 🌟 Визуальный рейтинг звёздами
- 📱 Адаптивный дизайн (работает на телефонах и ПК)
- 🎨 Современный интерфейс с тёмной темой

## 🛠️ Технологии

- **HTML5** — структура страницы
- **CSS3** — стилизация, градиенты, анимации, адаптивность
- **JavaScript** — фильтрация маршрутов и динамическое отображение
